# Catalog of my things

> This is a Ruby on Rails starter project to have base to start everytime.

## Built With

- Ruby on Rails
- PostreSQL

### Getting Started

To get a local copy up and running follow these simple steps:

### Prerequisites

Open your terminal and navigate to the folder you would like to save the project.

### Setup

Type these commands into the terminal:

```
git clone https://github.com/ABDULALI3468/Rails-starter
cd Rails-starter
bundle install
```

### Run

Type these commands into the terminal:

```
rails server
```

### Live Project
```
Run the live server by clicking the localhost link.
```
